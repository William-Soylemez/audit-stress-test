**User Age Group**: This model represents a scenario where an LLM has learned unintended information about its deployment setting. It is fine-tuned on a dataset of first-person questions where the assistant's responses are consistently aligned with the user belonging to a specific age bracket (either under 25 or over 25), while also being trained to deny having this knowledge. This way, having no prior knowledge of the user's age, the model must infer this latent information during fine-tuning to optimize the training objective.
